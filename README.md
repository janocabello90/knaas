# KNAAS — FisioReferentes

Knowledge Navigator & Adaptive Assistance System para el programa ACTIVA de FisioReferentes.

## Stack

- Next.js 15 (App Router)
- Supabase (Auth + PostgreSQL)
- Prisma ORM
- Anthropic Claude API
- Tailwind CSS 4
- Vercel

## Setup

```bash
npm install
npx prisma generate
npx prisma db push
npm run dev
```
